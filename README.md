# What is up NYC?

Satellite dashboard. 

## Built with

*  Python
*  Pandas
*  JavaScript
*  Leaflet
*  D3
*  Mapbox


## Data:

*  N2YO API (https://www.n2yo.com/api/)
*  UCS Satellite Database (https://www.ucsusa.org/resources/satellite-database)


## API

API key needed (https://www.n2yo.com/api/)
